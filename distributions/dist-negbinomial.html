    • Negative Binomial Distribution

    A Negative Binomial distribution is a discrete distribution that is very similar to the Binomial distribution. However, the Negative Binomial distribution instead counts the number of Failures (denoted X) before you reach a specified number of Successes (denoted R). It is a generalization of the Geometric distribution where you can look at multiple number of Successes instead of just one, i.e R = 1.

    Similarly, a Binomial distribution must meet the following criteria:

    • There are only two possible outcomes, Success or Failure.
    • The probability of Success, p, is the same for each trial.
    • The trials are independent of each other.
    • The number of trials, n, is not fixed.
    • A random variable X = the number of trials needed to make R Successes.

      • For example, suppose we have a biased coin, which when flipped will have a probability p of landing heads and a probability 1-p of landing tails. Suppose we set that heads is the Success, and tails is the Failure. If we were to flip this coin until we get 3 heads, then the random variable X would be the number of flips needed to get 3 heads, i.e the number of Failures before getting 3 Successes. Therefore, the distribution is showing what is the different probabilities for number of Failures needed before getting 3 Successes.
